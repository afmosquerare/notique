# Notique - Official Landing Page & Playground

This directory contains the source code for the official landing page and interactive playground of **Notique**.

Built with:
- [Astro](https://astro.build/) - Web framework
- [Tailwind CSS v4](https://tailwindcss.com/) - Styling
- [DaisyUI](https://daisyui.com/) - UI Components
- [Notique](https://github.com/afmosquerare/notique) - The underlying sub-2kb notification library

## Development Commands

All commands are run from the root of this `website` directory:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:5173`      |
| `npm run build`           | Builds your production site to `./dist/`         |
| `npm run preview`         | Previews your build locally                      |
